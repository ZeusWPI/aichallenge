Compile en run command?

We laten je toe een resem files up te loaden die je bot moet voorstellen.
Nu moeten wij wel weten hoe we je bot in gang steken, ziedaar het run commando.
Gebruik je een taal die gecompileerd moet worden?
Geef het compileercommando ook mee!
Als je wil, kan je zelfs een makefile mee uploaden, en gewoon de makefile laten uitvoeren in je commando.
